# esp32-motor-driver
BTS7960 High Power Motor Driver ESP32 Test

### BTS7960 Connections

- B+ - 5.5 - 27V
- B- - Gnd
- M+ - Motor Positive
- M- - Motor Negative
- VCC - 3V3
- GND - GND
- R_EN - 3V3
- L_EN - 3V3
- RPWM - GPIO 32
- LPWM - GPIO 33
